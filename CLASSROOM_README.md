# Git Practice: Push, Pull, and Merge Conflicts

Welcome to the Git collaboration exercise! This repository will help you practice working with remotes and handling merge conflicts.

## Instructions

Follow the exercise instructions provided in class. You'll be:

1. Making your first commit and pushing to remote
2. Pulling automated changes (simulating a teammate's work)
3. Creating and resolving a merge conflict

## Files

- `some_arithmetic.py` - Your practice file for this exercise

## Important Notes

⚠️ This repository uses **GitHub Actions** to simulate collaborative scenarios. After you push, wait about 30 seconds before pulling to allow the automation to run.

🎯 **Learning Goal**: By the end of this exercise, you'll be comfortable with the push-pull workflow and confident in resolving merge conflicts!
